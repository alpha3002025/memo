## google youtube Java,Kotlin library

google 에서 제공해주는 라이브러리들은 아래와 같습니다.

```kotlin
dependencies{
	implementation("com.google.api-client:google-api-client:1.23.0")
	implementation("com.google.oauth-client:google-oauth-client-jetty:1.23.0")
	compileOnly("com.google.apis:google-api-services-youtubeAnalytics:v2-rev16-1.23.0")
	implementation("com.google.apis:google-api-services-youtube:v3-rev212-1.25.0")
    // ...
}
```

이 라이브러리들은 직접 사용해본 결과 특정 Response 의 경우 데이터 타입이 불일치가 발생하고, 인증기능을 사용할 때에는 redirect uri mismatch 에러가 나지만, 실제 에러가 발생한 원인은 엉뚱한 원인인 경우도 있고, 라이브러리 자체가 불안정합니다.<br/>



따라서 가급적이면 Java/Kotlin 을 사용해서 애플리케이션을 개발할 것이라면 HTTP/Curl 명세를 직접 호출해보고, 잘 동작하면 그 API 호출을 Feign Client 또는 WebClient 기반으로 전환하시길 권장드립니다.<br/>
