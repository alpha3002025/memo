# python 의 연산자

산술연산

- `+` : 더하기
- `-` : 뺄셈
- `*` : 곱셈
- `/` : 나눗셈
- `//` : 몫
- `%` : 나머지 연산
- `**` : 제곱연산

<br/>



비교연산

- `>` : 크다
- `<` : 작다
- `>=` : 크거나 작다
- `<=` : 작거나 같다
- `==` : 같다
- `!=` : 다르다

<br/>



논리연산

- `A and B` : A, B 모두 참일때
- `A or B` : A, B 둘중 하나라도 참일때
- `not A` : A 가 참이아닐때

<br/>



멤버십 연산

- `in` : 포함되어있는지
- `not in` :포함이 안되어 있는지
